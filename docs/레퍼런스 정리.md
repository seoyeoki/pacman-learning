# 1. [논문 분석] 인공신경망과 유전 알고리즘을 이용한 팩맨 게임

## 1. 배경 지식 (Prerequisites)
이 논문을 구현하기 위해서는 다음 세 가지 핵심 개념을 이해해야 합니다.

* **유전 알고리즘 (Genetic Algorithm, GA):**
    * 자연의 진화 과정을 모방한 최적화 알고리즘입니다.
    * [cite_start]**과정:** 초기 무작위 개체 생성 -> 우수한 개체 선택(Selection) -> 교배(Crossover) & 변이(Mutation) -> 다음 세대 생성[cite: 32].
    * 이 프로젝트에서는 "가장 오래 살아남은 팩맨"이 자손을 남기게 됩니다.
* **인공신경망 (Artificial Neural Network, ANN):**
    * 팩맨의 **'뇌'** 역할을 합니다. 센서값(입력)을 받아 이동할 방향(출력)을 결정합니다.
    * [cite_start]일반적인 딥러닝과 달리 여기서는 '역전파(Backpropagation)'로 학습하지 않고, 유전 알고리즘을 통해 **가중치(Weight) 자체를 진화**시킵니다[cite: 46].
* **상태 인식 (State Sensors):**
    * 화면 전체(픽셀)를 보는 것이 아니라, 팩맨 기준으로 상하좌우에 무엇이 있는지를 감지하는 단순화된 센서를 사용합니다.

## 2. 논문 핵심 구현 명세 (Technical Specs)

### A. 팩맨의 눈 (Input Features)
논문은 픽셀 전체를 입력받지 않고 **간소화된 센서**를 사용합니다. 이는 학습 속도를 높이는 핵심 요인입니다.
* [cite_start]**센서 방식:** 팩맨 위치에서 진행 가능한 방향으로 탐색을 진행하여 유령을 감지합니다[cite: 109].
* [cite_start]**탐색 범위:** 두 번째 교차로(갈림길)까지만 탐색합니다[cite: 116, 118].
* **입력 데이터 형태:** `[우, 좌, 상, 하]` 4개의 값을 가진 배열. (예: 오른쪽에 유령이 있으면 `[1, 0, 0, 0]`) [cite_start][cite: 112, 113].

### B. 팩맨의 뇌 (Neural Network Architecture)
[cite_start]입력과 출력 사이를 연결하는 신경망 구조입니다. [cite: 127]
* **입력층 (Input Layer):** 노드 4개 (유령 센서 값)
* **은닉층 (Hidden Layers):** * 1층: 10개
    * 2층: 20개
    * 3층: 10개
    * [cite_start]활성화 함수: **ReLU** 사용[cite: 139].
* **출력층 (Output Layer):** 노드 4개 (이동할 방향: 상, 하, 좌, 우).
    * [cite_start]활성화 함수: **Softmax** 사용 (확률값으로 변환)[cite: 143].
* [cite_start]**판단:** 출력값 중 가장 큰 값을 가진 방향으로 이동[cite: 149].

### C. 진화 전략 (Evolution Strategy)
어떻게 팩맨을 똑똑하게 만들 것인가에 대한 규칙입니다.
* [cite_start]**개체군 (Population):** 한 세대당 **16마리**의 팩맨 AI 생성[cite: 120, 156].
* [cite_start]**적합도 (Fitness Function):** * 점수(포인트)보다 **생존(유령 회피 횟수/시간)**을 우선순위로 둡니다[cite: 164].
    * [cite_start]유령이 있는 방향으로 가면 감점, 피하면 가점[cite: 165].
* **선택 및 교배:**
    * [cite_start]상위 4개의 우수 유전자를 선택[cite: 166].
    * 랜덤하게 가중치 행렬을 섞어(Crossover) 자식 생성.
* **변이 (Mutation):**
    * [cite_start]지역 최적점 탈출을 위해 **20% 확률**로 가중치를 임의 변경[cite: 169].

## 3. 구현 로드맵 수정 제안
이 논문을 따르기 위해 기존 로드맵을 다음과 같이 조정해야 합니다.

1.  **Phase 1 (기존 완료):** 맵 생성 및 기본 이동.
2.  **Phase 2 (센서 구현 - 변경됨):** * 유령(Ghost)을 단순하게 추가.
    * 논문에서 묘사한 **"교차로 기반 4방향 유령 탐지 센서"** 구현.
3.  **Phase 3 (뇌 & 진화 구현 - 변경됨):**
    * PyTorch나 TensorFlow 없이 **NumPy만으로** 간단한 신경망(MLP) 구현.
    * 16마리 팩맨을 돌리고 평가하는 **유전 알고리즘 루프** 구현.
